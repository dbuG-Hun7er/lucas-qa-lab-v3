Como usar os leitores HTML

Os arquivos HTML abrem direto no navegador. Para leitura em inglês, você pode usar o Google Tradutor do Chrome.

  1. Abra o arquivo HTML no Google Chrome.
  2. Clique com o botão direito em qualquer área da página.
  3. Escolha Traduzir para português.
  4. Se necessário, ajuste o idioma no ícone do Google Tradutor.
  5. Use o modo dark e a navegação interna do leitor para estudar no celular.
